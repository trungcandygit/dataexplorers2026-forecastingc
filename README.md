## Mục tiêu

Xây dựng mô hình dự báo nhu cầu cho **Thống Nhất Bike** (B2B), trả lời 3 câu hỏi dự báo cho **Q2/2026**:

| Câu hỏi | Notebook | Mô hình |
|---------|----------|---------|
| **C1**: Sản phẩm nào sẽ bán được? | `C1_Forecast_Sales_Q2_2026.ipynb` | Prophet + SARIMAX |
| **C2**: Màu sắc nào sẽ được ưa chuộng? | `C2_Forecast_Color_SKU.ipynb` | Prophet per color |
| **C3**: Đại lý nào sẽ mua hàng? | `C3_Forecast_Dealer_Activity.ipynb` | GBM Classifier (RFM) |
| **Bonus**: LLM Q&A on forecast | `Ensemble/C_Ensemble_LLM.ipynb` | Ensemble + OpenAI |

├── Ensemble/

## Setup môi trường

```bash
# Tạo conda environment
conda create --name forecasting python=3.8
conda activate forecasting

# Cài dependencies
conda install -c conda-forge prophet
pip install pmdarima statsmodels scikit-learn xgboost plotly ipywidgets sqlalchemy psycopg2-binary openai jupyter

# Chạy Jupyter
jupyter notebook
```
